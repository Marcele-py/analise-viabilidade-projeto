Este projeto tem como objetivo avaliar a viabilidade econômica de um investimento utilizando funções financeiras do Excel. Foram aplicados conceitos como Valor Presente (VP), Valor Presente Líquido (VPL). Taxa Interna de Retorno (TIR), payback simples e descontado.
Os cálculos foram desenvolvidos de duas formas: Manualmente, por meio da estruturação de tabelas e utilizando as funções financeiras do Excel, demonstrando domínio tanto dos conceitos, quanto da ferramenta.
O projeto conta com organização de dados, tabelas dinâmicas, gráficos e um dashboard para facilitar a visualização e a análise dos resultados, auxiliando na tomada de decisão sobre a viabilidade do projeto.
Ferramentas utilizadas: Excel, fórmulas financeiras, tabelas dinâmicas, gráficos e dashboard.
Os dados apresentados nesse projeto não exibem informações sensíveis de quaisquer clientes.
